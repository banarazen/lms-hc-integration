Interactive Walkthrough
Key Integration Features (v1 Scope)
💬 Two-Way SMS
Send/receive SMS from LMS using HyperConnect's messaging component. Bidirectional sync within 1 second.
📞 Voice Calling
Manual outbound calls from LMS using the production-ready Flutter voice package. Full call controls.
🤖 AI Calling
Schedule AI outbound calls with prompts, context, and business hours compliance. Bulk 100+ parallel calls.
🏷 Lead Badges
Visual LEAD indicator in HyperConnect. Shows count for multiple leads. Removed on conversion.
🔁 Conversation Sync
SMS, calls, voicemails sync bidirectionally. Tasks, notes, stage changes stay in LMS only.
🔗 Cross-Navigation
HC → LMS deep link via lead events. LMS → HC via linked guest profile. App-switch pattern.
🌳 Expandable Tree
Multiple leads per person with expand/collapse. Consolidated (person) vs filtered (lead) timeline views.
🏆 Lead Conversion
Convert lead → badge removed in HC → timeline archived in LMS → guest profile continues.
🔢 Event Filtering
"All events" dropdown in LMS timeline. Filter by Messages, Calls, Tasks, Notes, Lead Events.
